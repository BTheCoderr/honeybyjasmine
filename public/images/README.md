# Image Assets for HoneyByJasmine

## Image Structure

Place your images in the following directories:

### `/images/hero/`
- **hero-main.jpg** (or .png, .webp) - Main hero section food photo
  - Recommended size: 1200x1200px (square) or 1600x900px (landscape)
  - Format: WebP or JPG
  - Should showcase your best meal prep dishes

### `/images/meals/`
Meal photos for the weekly menu preview section:
- **salmon-bowl.jpg** - Honey Glazed Salmon Bowl
- **chicken-wrap.jpg** - Mediterranean Chicken Wrap
- **veggie-bowl.jpg** - Veggie Power Bowl
- **turkey-meatballs.jpg** - Turkey Meatball Marinara
- **thai-curry.jpg** - Thai Curry with Jasmine Rice
- **bbq-chicken.jpg** - BBQ Pulled Chicken Bowl

Recommended size: 800x600px (4:3 ratio)
Format: WebP or JPG

### `/images/about/`
- **jasmine-photo.jpg** - Jasmine's professional photo
  - Recommended size: 800x800px (square) or 600x800px (portrait)
  - Format: WebP or JPG
  - Should be warm, welcoming, and professional

## Image Optimization Tips

1. **Use WebP format** for better compression and quality
2. **Optimize file sizes** - aim for under 200KB per image
3. **Use descriptive filenames** - helps with SEO
4. **Maintain aspect ratios** - prevents image distortion
5. **Consider lazy loading** - already implemented in the code

## Placeholder Images

Until real images are added, the site will display gradient placeholders. Replace the placeholder divs with `<img>` tags once you have your images ready.
